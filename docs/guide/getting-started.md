# Getting Started

Wardove is a personal wardrobe tracker for Android. It lives entirely on your device — no account, no cloud, no subscription.

## Install

1. Go to the [Releases page](https://github.com/zorenkonte/wardove/releases) on GitHub.
2. Download the latest `app-release.apk`.
3. On your Android device, allow installs from unknown sources (Settings → Apps → Special app access → Install unknown apps), then open the APK to install.

::: tip Minimum Android version
Wardobe requires **Android 7.0 (API 24)** or newer.
:::

## First launch

On first launch Wardove shows an empty wardrobe. Tap the **+** button to add your first item — you'll need a photo (camera or gallery), a name, a category, and a color to save it.

## In-app updates

Wardove can update itself without an app store. From **Settings → About → Check for updates** the app queries GitHub Releases, and if a newer version is available you can download and install it directly. The same keystore is used for every release, so the update installs cleanly over the previous version.

## Data & privacy

All your data — wardrobe items, wear logs, laundry history, and photos — is stored **locally in the app sandbox**. Nothing leaves your device. Uninstalling the app deletes everything.

There is no cloud sync. Backing up your data requires a full device backup (e.g. Android's built-in backup or ADB).
