# Calendar

The Calendar screen shows a monthly calendar of your wear history.

## Usage

- Days with at least one wear log are **marked** on the calendar.
- Tap a date to see which items you wore that day.
- The calendar opens on **today** by default.
- Navigate forward and backward through months using the arrow controls.

## Wear logs

Each entry in the day view corresponds to a single clothing item worn on that date. The same item can be worn on multiple days; each day gets its own log entry.

Wear logs are created from the [Item Detail](./item-detail#wear-today--un-wear-today) screen.
