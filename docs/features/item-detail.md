# Items

## Adding an item

Tap the **+** button on the Wardrobe screen to open the Add Item screen.

Required fields:

| Field | Notes |
|-------|-------|
| Photo | Take a new photo with the camera, or pick from gallery |
| Name | Free-text label |
| Category | Top, Bottom, Shoes, Outerwear, or Accessory |
| Color | One of 12 preset colors |

Optional fields: **Notes**, **Price** (used for cost-per-wear in Stats).

Tap **Save** when all required fields are filled. The item is created with status **Clean**.

## Editing an item

Open an item's detail screen and tap **Edit**. All fields (including the photo) are editable. The same required-field rules apply.

## Item Detail

The detail screen shows:

- Full photo
- Category, color, notes, price
- Current status badge
- Full wear-log history (dated list of every wear)

### Wear today / Un-wear today

Tap **Wear today** to log a wear for the current date. The item's status changes to **Worn** and its total wear count increments.

If you tapped by mistake, tap **Un-wear today** to remove today's log and revert the status.

::: info
You can only log or un-log a wear for **today**. Past wear logs are not editable.
:::

### Deleting an item

Tap **Delete** on the detail screen. This removes the item, all its wear logs, and its photo from storage. The action is not reversible.
